# Vennela Makeover Studio Website

A beautiful, elegant website for Vennela Makeover Studio - Hyderabad's premier bridal makeup artist.

## 🚀 Quick Start

### Deploy to Netlify (Easiest - 30 seconds)
1. Go to [app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag and drop this entire folder
3. Your site is live!

### Deploy to GitHub Pages (Recommended)
See setup instructions below.

## 📁 Project Structure

```
vennela-makeover-studio/
├── index.html          # Main website file
├── images/             # Add your images here
│   ├── about/          # About section photo
│   ├── gallery/        # Portfolio images
│   └── hero/           # Hero background (optional)
└── README.md           # This file
```

## 🖼️ Adding Images

### About Photo
Replace the placeholder in `index.html`:
```html
<!-- Find this line and uncomment, add your image -->
<img src="images/about/vennela-photo.jpg" alt="Vennela - Makeup Artist">
```

### Gallery Images
Replace gallery placeholders:
```html
<div class="gallery-item">
    <img src="images/gallery/bridal-1.jpg" alt="Bridal Makeup">
    ...
</div>
```

### Recommended Image Sizes
- About photo: 600x800px (portrait)
- Gallery: 800x1000px (portrait) or 1200x800px (landscape)
- Compress at [tinypng.com](https://tinypng.com)

## 📧 Enable Contact Form

1. Sign up at [formspree.io](https://formspree.io) (free)
2. Create a new form
3. Copy your form ID
4. Replace `YOUR_FORM_ID` in index.html

## 🌐 Custom Domain

1. Buy domain (~₹800/year): namecheap.com or godaddy.com
2. In Netlify: Site Settings → Domain Management
3. Add your domain and configure DNS
